# Personal Assistant Project

This project is a Python-based personal assistant capable of responding to voice commands, scheduling tasks, and retrieving online information, such as real-time weather data from any location.

## Features
- Voice interaction (currently under development
- Add and manage tasks 
- Real-time weather information for any location
- Basic reminders and scheduling system
- Text-to-speech functionality
## Next Steps:
- Implement speech recognition, 
- Visual reprentation using liberay like Tkinker 
- Integrate natural language processing for better user interaction.
## How to run
1. Install the required libraries using pip install -r requirements.txt (if you have one).
2. Clone the repository.
3. Run the `personal_assistant.py` file.
4. Follow the on-screen prompts to interact with the assistant.
